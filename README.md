# Salon CRM Appointment Booking System

A modern Salon CRM (Customer Relationship Management) web application built using Angular. The system helps salon owners manage appointments, customers, staff, services, billing, and revenue tracking from a centralized dashboard.

## 🚀 Live Demo

https://salon-crm-sigma.vercel.app

## 📌 Features

* Secure Login Authentication
* Dashboard with Business Insights
* Appointment Booking & Management
* Customer Management
* Staff Management
* Service Management
* Billing & Invoice Generation
* Revenue Tracking
* Responsive User Interface
* Local Storage Data Persistence

## 🛠️ Tech Stack

### Frontend

* Angular 20
* TypeScript
* HTML5
* CSS3

### Tools & Deployment

* Git
* GitHub
* Vercel

### Storage

* Browser LocalStorage
